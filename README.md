<h1>Setup</h1>
<p>Make sure you have a .env file with your email and password in it!<br>
Use MICROSOFT_EMAIL and MICROSOFT_PASSWORD.<br>
You may need to use Microsoft Authenticator for 2FA, in this case please switch to manual-auth branch.<br>
It is not robust AT ALL so for it to work use a stable internet connection (not mobile data)</p>
